# Firmware ESP32

Questa cartella contiene i firmware del progetto GGNO-IOT.

## Struttura prevista
- full/    → firmware completo (cancello + caldaia)
- gate/    → modulo apertura cancello (SIM800L + Telegram)
- boiler/  → modulo gestione caldaia / termostato
